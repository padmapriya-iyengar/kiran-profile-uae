# Kiran Joshi Profile UAE

Static professional profile website for Kiran Joshi, built for cloud hosting and profile sharing in the UAE.

## Files

- `index.html` - complete single-page web application
- `styles.css` - responsive visual design
- `script.js` - mobile navigation and active section state
- `assets/enterprise-dubai-ai-hero.png` - generated hero visual
- `assets/KiranJoshi_Enterprisetech_Leader_UAE.pdf` - downloadable profile

## Run Locally

Open `index.html` directly in a browser, or serve the folder with any static server.

```powershell
python -m http.server 4173
```

Then visit `http://localhost:4173`.

## Deploy

Upload the full folder to any static host such as Azure Static Web Apps, Netlify, Vercel static output, AWS S3 + CloudFront, GitHub Pages, or Cloudflare Pages.
